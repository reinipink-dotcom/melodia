### Task 2 — Create src/utils/audioPlayer.ts

**File:** `src/utils/audioPlayer.ts` (new file)

Create a module that wraps `expo-av` Sound with expo-speech fallback. Export two functions:

**`playTrigger(trigger: TtsTrigger): Promise<void>`**
- Calls `stopAudio()` first to unload any playing sound.
- Constructs a `require()` path from `trigger.outputFile` using a static lookup map (see below — expo-av `require()` calls must be statically analyzable at build time, so a switch/map of all 17 known files is required; dynamic `require()` does not work with Metro bundler).
- Calls `Audio.Sound.createAsync(source, { shouldPlay: true })` and stores the Sound object in a module-level ref.
- On any error (file not found, load failure), catches the exception and calls `speakSpanish(trigger.text)` as fallback — no rethrow.

**`stopAudio(): void`**
- Calls `sound.unloadAsync()` on the stored Sound ref if it exists, then sets ref to null.
- Does not throw — wraps in try/catch.
