# Melodia — Obsidian Communication Layer

This vault is how Reine passes notes, feedback, and context to Claude between sessions.

## How it works

**One file: `inbox.md`**

Write anything there — feedback on screens, bugs you found, ideas, questions, design references, whatever. Claude reads it at the start of each session (when you say "check Obsidian"), processes everything, then clears it.

**Images: drop in `design/`**

If you want Claude to look at a screenshot or design inspiration, drop the image file in the `design/` folder and mention it in inbox.

That's it. No other folders needed.

## What to write in inbox

- UI feedback ("the quiz screen colors felt off")
- Bugs you found ("the continue button doesn't work")
- Design ideas or references ("I saw this app and loved how they did X")
- Questions ("should we do X or Y for Phase 5?")
- Phase review notes before a session
- Literally anything — don't organize it, just write
