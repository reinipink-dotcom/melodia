# Reference: Playful Mascot Language Learning Onboarding
Source image: 01-onboarding-playful-mascot-language.png

## What to borrow for Melodia

### Overall Mood
- Bold, high-contrast color blocks per screen (yellow, lavender, orange) — each onboarding screen has its own personality color
- Cute illustrated mascot characters (rabbit, blob, alien-like figure) occupy the lower 40% of each screen
- Very minimal text — one large headline, one short subline, one CTA button
- Feels playful and confident, not corporate

### Onboarding Screen Structure
- Full-screen color background (not dark — but the layout pattern applies to dark mode)
- Large bold headline at top-left, 2–3 lines, ~32px, heavy weight
- Mascot illustration centered/lower half — takes up significant vertical space
- Single pill CTA button at very bottom: "Get Started »"
- Dot pagination indicators above the button

### Typography
- Headlines: very large, bold, left-aligned, high contrast on the color bg
- Inline tag/pill ("Languages 🔵") embedded within the headline text — interactive feel
- No body text clutter — trust the illustration to carry meaning

### What This Means for Melodia
- Melodia's onboarding (screens 1–7) should use the Melodia mascot (coral music note) as the visual hero on each screen, not just decoration
- Each onboarding screen could have a distinct accent color wash or gradient variation to give each step its own feel
- The "Lock in" commitment screen (screen 7) is a perfect candidate for this full-mascot treatment
- Keep text minimal — the mascot + one bold line does more work than paragraphs
- Pill CTA at bottom with progress dots is the right pattern for Melodia's onboarding nav
