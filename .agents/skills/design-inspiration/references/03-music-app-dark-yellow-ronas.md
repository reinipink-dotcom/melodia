# Reference: Music App UI — Ronas IT UI/UX Team
Source image: 03-music-app-dark-yellow-ronas.png

## What to borrow for Melodia

### Overall Mood
- Very dark background (#111111 near-black) with a single vivid accent: neon yellow (#F5FF00)
- Minimal chrome — the UI disappears and the music content is the hero
- Album art is large and dominates every screen
- Feels premium, editorial, and music-first

### Discover / Home Screen
- "Discover" section header large and bold, top left
- Full-bleed album art card with genre label overlay ("Trap") at bottom-left, play button centered
- Track list below: thumbnail + artist + track name + duration, simple rows
- Bottom nav: minimal icon-only, 3 items

### Now Playing Screen
- Full-screen artist photo (portrait orientation, high contrast)
- Song title + album overlaid at bottom on semi-transparent dark panel
- Waveform visualization replaces traditional progress bar — very distinctive
- Transport controls: minimal, icon-only, centered row
- Back arrow top-left only nav element

### Recommended Screen
- Search bar: dark pill shape, placeholder text, no border
- "Recommended For You Today" section header
- Track rows: timestamp shown instead of duration (11 PM) — context-aware
- "New Collection" horizontal scroll of square art cards

### Color Application for Melodia
- Yellow accent → swap for Melodia's Coral #FF6B4A
- Near-black bg → Melodia's Midnight #1A1A2E
- This entire color system translates 1:1 to Melodia's palette

### What This Means for Melodia
- Listen screen (screen 13): large album art card, full-bleed, song info overlaid at bottom
- The waveform as progress indicator is a beautiful pattern for the listen/quiz transition
- Track list rows (home + playlist view): thumbnail + info + right-side metadata
- Keep bottom nav minimal — icon only, no labels needed once users learn the app
