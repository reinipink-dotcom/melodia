## How to add new references

1. Drop your screenshot into this folder
2. Name it: `NN-short-description.png` (e.g. `10-paywall-dark-duolingo.png`)
3. Claude will read both the image and generate a `.md` analysis the next time `/design-inspiration` runs

## Naming convention
```
NN-topic-style-source.png
```
