# Reference: PULSE — Sport Music App Branding (Admiral Studios)
Source image: 04-music-branding-pulse-orange.png

## What to borrow for Melodia

### Overall Mood
- Deep black + burnt orange (#E8530A range) — warm, energetic, premium
- Branding grid layout: each tile is a standalone brand moment
- Photography style: dramatic, high-contrast, moody portraits — cinematic

### Brand Grid Composition
- Portrait photo tile: subject slightly blurred/motion, small app icon overlaid bottom-left
- Hero statement tile: large text on gradient bg ("Music for muscles and mind.")
- Logo lockup tile: wordmark on white, clean and simple
- Stat tile: metric + label on muted gray background (1.3x / FASTER HEART-RATE RECOVERY)
- Feature tile: key spec as large number + label + play button (#120ms / ADAPTIVE RESPONSE)
- Waveform tile: audio waveform bars on dark bg — brand texture
- Icon grid tile: 4 category icons in a 2×2 grid

### App Icon Treatment
- Square icon with small flame/spark symbol — simple, geometric
- Works on both dark and light backgrounds

### Stat Display Pattern (directly applicable to Melodia achievements)
- Large number (1.3x) dominant
- Label below in small caps or light weight
- Muted background so the number pops
- Maps to Melodia's XP / streak / score displays

### Feature/Spec Tile Pattern
- Metric in accent color, large
- Short label in caps below
- Play/action button to the right
- Maps to Melodia's module detail screen: level badge + concept + "Start lesson"

### What This Means for Melodia
- Achievement screen (screen 22): use this branding grid layout for milestone cards
- App Store screenshots: this grid composition style looks incredible in marketing materials
- Module detail stats (XP, duration, quiz count) displayed as these bold stat tiles
- The orange palette is extremely close to Melodia's coral — this whole system is reference-ready
