# Reference: Music App Branding — Halo Lab / Maunsound
Source image: 04-music-branding-orange-waveform.png

## What to borrow for Melodia

### Brand System Approach
- Waveform as the primary brand mark — the waveform icon works across all colors/backgrounds
- Modular brand grid: each tile is a self-contained moment (logo on black, logo on blue, stats tile, photo tile, tagline tile)
- Strong typographic statements as standalone brand assets: "SOUND HAS A CLEAR PURPOSE", "SOUND IS PART OF DESIGN"

### Player Card (top left tile)
- Compact now-playing widget: artist photo thumbnail (small, rounded) + track name + artist + progress bar + transport controls
- Dark background with gradient wash (black to orange/red)
- Progress bar: blue fill — maps to Melodia's teal or coral
- Very tight, information-dense layout — good reference for Melodia's mini-player or listen screen card

### Typography Style
- All-caps, heavy weight, wide tracking for brand statements
- Mix of serif and sans for visual interest in long-form contexts
- Numbers (87%) shown very large and bold as a stat highlight

### Color Philosophy
- Black + one vivid accent (orange/red) — the same formula as Melodia's Midnight + Coral
- Blue used as a secondary accent for interactive elements (progress bar, active button)
- Pure orange fields used for artist/album card backgrounds when no photo is available

### Icon / App Icon Treatment
- Music note inside a rounded square — simple, recognizable
- Works in coral/orange on dark and dark on light

### What This Means for Melodia
- Melodia's mascot (coral music note) follows this exact icon convention — validated
- The "one vivid accent on black" system is proven across multiple references — commit to it fully
- Typography for achievement/celebration screens can go bold + all-caps + large for impact
- Mini-player widget layout (thumbnail + title + artist + scrubber + controls) is a clear pattern
