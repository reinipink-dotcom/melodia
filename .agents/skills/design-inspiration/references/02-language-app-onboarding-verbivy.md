# Reference: Verbivy — Language Learning App (Muhammad Dani Asyrofi / Kretya Studio)
Source image: 02-language-app-onboarding-verbivy.png

## What to borrow for Melodia

### Onboarding Flow Structure
- Welcome screen: full illustration top half, greeting text + user name bottom half — personal and warm
- Daily goal screen: time options as pill chips in a grid (15 min / 30 min / 45 min / 60 min / Custom) — selected state filled dark, unselected outlined
- Lesson categories screen: toggle filter chips (Daily Conversations / Business English / English for Travel / Listening Comprehension)
- Lesson list: card per lesson with illustration + title + subtitle + "Start Now" badge
- "You're All Set!" confirmation screen before entering app
- Lesson Complete screen: celebration message + confetti treatment

### Daily Goal Picker (directly applicable to Melodia screen 6)
- Pills laid out in a 2×3 grid
- Selected pill: dark filled background, white text
- Unselected pill: light border, dark text
- Custom option included at end
- Single "Next" CTA below the grid

### Lesson Card Style
- Horizontal card: illustration left, text right
- Title bold ~16px, subtitle muted ~13px
- "Start Now" badge in accent color (green) — could be coral for Melodia
- Subtle shadow under card, rounded corners ~12px

### "All Set" / Confirmation Screen
- Centered layout: icon/illustration top, bold celebration headline, supporting text, single CTA
- Maps directly to Melodia's "Lock in" commitment screen (screen 7)

### Lesson Complete Screen
- "Good job!" header with confetti/sparkle treatment
- Module title confirmation
- Single continue CTA
- Maps to Melodia's lesson complete screen (screen 17)

### What This Means for Melodia
- Daily goal picker: use this exact pill grid pattern for screen 6
- Module list cards: illustration/icon left + title + level badge right
- Lesson complete: simple centered celebration, don't overcomplicate it
- "You're All Set" pattern: use between commitment screen and account creation
