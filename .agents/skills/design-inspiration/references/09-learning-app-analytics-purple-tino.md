# Reference: Personalized Learning Progress (Tino App)
Source image: 01-learning-app-analytics-purple.png

## What to borrow for Melodia

### Layout & Structure
- Home screen uses a large greeting ("Hi 👋 Ben") with bold weight — warm, personal feel
- Featured module card is wide/full-bleed with progress bar at bottom (88% shown) — great for Melodia's current module card
- Horizontal scrollable filter pills (All / Popular / Design) — directly applicable to Modules screen genre filter
- Course cards are 2-column grid with thumbnail image, title, instructor, and time tracked below
- Analytics screen uses a dark card embedded in a light screen — striking contrast technique

### Analytics / Stats Cards
- Streak and Points shown as two side-by-side colored cards (pink for streak, blue for points) with icon + big number + subtext
- Week challenge shows a 7-day calendar row with filled circles for completed days — perfect for Melodia streak tracker
- Trending section uses icon + label + hours + points + circular progress ring

### Colors
- Primary accent: soft purple (#7B61FF range)
- Backgrounds: white / very light gray (light mode) — note: Melodia is dark mode, invert the contrast logic
- Success/active states: filled purple circles
- Inactive/future: light gray circles

### Typography
- Section headers: medium weight, ~16–18px
- Big numbers (stats): bold, ~32–40px
- Supporting labels: light weight, ~12px, muted gray

### Components to Replicate
- Progress bar under module card (thin, accent-colored, rounded)
- Week calendar row with filled/unfilled day indicators
- Side-by-side stat cards with colored backgrounds and icon in corner
- Horizontal pill filter row (scrollable)
